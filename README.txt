Гайд по настройке бота

Онлайн:
1. В config.py: MODE = ONLINE
2. Заполнить on_point, off_point, on_reaction, off_reaction по шаблону из models.py
3. Загрузить все файлы в лс бота, в data/file.txt будут их file_id. Дописать к ним необходимые поля в этом файле по шаблону
    После последней строчки данных на новой строке должен стоять символ .
4. Запустить fill_script.py
5. Отметить номера последних онлайн и оффлайн кп в config: LAST_ON_POINT_NUM и LAST_ON_POINT_NUM
6. Создать беседы
7. В чате инициализоровать команды-чаты: 
	/reg название кол-во_участников
8. Ввести /pinned и закрепить
9. удалить все сообщения кроме pinned
10. Косметика беседы - название, ава и тд
11. Разрешить в чатах чтение ботом сообщений через BotFather: /setprivacy
12. Добавить участников
13. Готово к работе
14. Для ввода ответа сначала надо ввести команду /answer, затем обычным текстом ответ
15. После прохождения онлайн-части текущей командой, доступ к игре будет у нее закрыт

Оффлайн:
1. В config.py поменять: MODE = OFFLINE_MODE
2. /flush в лс с ботом
3. Команда - /kill в беседах с участниками(могут и должны ввести сами участники)
4. Вводить /checkin код_начала для начала КП, /checkout код_конца для конца КП.
5. После прохождения оффлайн-части текущей командой, доступ к игре будет у нее закрыт
